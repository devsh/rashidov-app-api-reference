# Получить личные данные :lock:

## Запрос

`GET /api/v1/personal`

Без параметров

### Пример
`GET /api/v1/personal`

## Ответ

| Параметр | Тип    | Описание          |
|----------|--------|-------------------|
| name     | string | Имя               |
| phone    | string | Телефон           |
| email    | string | Электронная почта |

### Пример
```JSON
{
    "ok": true,
    "name": "Андрей",
    "phone": "+7 (123) 456-78-90",
    "email": "mail@example.com"
}
```

## Ошибки
| Код ошибки | Описание                               |
|------------|----------------------------------------|
| 3          | Несуществующий токен                   |
