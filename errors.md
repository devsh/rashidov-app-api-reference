# Ошибки

```JSON
{
    "ok": false,
    "errorCode": 1,
    "errorDescription": "Wrong username"
}
```

| Код ошибки | Описание                                                |
|------------|---------------------------------------------------------|
| 1          | Неверный логин                                          |
| 2          | Неверный пароль                                         |
| 3          | Несуществующий токен                                    |
| 4          | Несуществующий токен обновления сессии                  |
| 5          | Неверный id                                             |
| 6          | Дата начала должна быть меньше или равна дате окончания |
| 7          | Недостаточно параметров                                 |
