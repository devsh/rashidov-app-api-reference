# Новое обращение :lock:

## Запрос

`POST /api/v1/issues/ticket`

| Параметр | Тип    | Описание        |
|----------|--------|-----------------|
| text     | string | Текст обращения |

### Пример

```JSON
{
    "text": "Добрый день!\n\nУ меня вопрос по весам..."
}
```

## Ответ

Без параметров

### Пример
```JSON
{
    "ok": true
}
```

## Ошибки
| Код ошибки | Описание                               |
|------------|----------------------------------------|
| 3          | Несуществующий токен                   |
