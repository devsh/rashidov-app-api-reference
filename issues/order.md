# Новый заказ :lock:

## Запрос

`POST /api/v1/issues/order`

| Параметр       | Тип    | Описание           |
|----------------|--------|--------------------|
| scalesAmount   | number | Количество весов   |
| stationsAmount | number | Количество станций |

### Пример

```JSON
{
    "scalesAmount": 1,
    "stationsAmount": 1
}
```

## Ответ

Без параметров

### Пример
```JSON
{
    "ok": true
}
```

## Ошибки
| Код ошибки | Описание                               |
|------------|----------------------------------------|
| 3          | Несуществующий токен                   |
