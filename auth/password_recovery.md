# Восстановление пароля

Отправляет новый пароль пользователю в sms.
## Запрос

`POST /api/v1/auth/password_recovery`

| Параметр | Тип    | Описание            |
|----------|--------|---------------------|
| username | string | логин пользователя  |

### Пример
```JSON
{
    "username": "username"
}
```

## Ответ

Без параметров

### Пример
```JSON
{
    "ok": true
}
```

## Ошибки
| Код ошибки | Описание             |
|------------|----------------------|
| 1          | Неверный логин       |
